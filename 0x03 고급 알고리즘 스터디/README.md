# ALPS 고급 알고리즘 스터디 2020

> 스터디 강사 : ALPS 노세윤 (컴퓨터학과 19)<br>스터디 일시 : 목요일 18시 30분 정보관

## 수업 목표

대회에서 사용되는 알고리즘을 익힙니다.
