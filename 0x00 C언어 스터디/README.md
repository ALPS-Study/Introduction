## 스터디 진행 방식

- 강의 시간 : 90분 예정
- PPT + vscode 사용하는 방식으로 설명
- 강의 이후 정리된 형태의 복습 자료 제공
- 백준 또는 쪽지 시험을 보는 식으로 스터디 이해도 및 진행 속도 점검 (필수 아님!)
- 유튜브에 업로드해 스터디 동영상 자료 남기기
- 스터디 이후 포스팅 형태로 남겨 아카이빙
