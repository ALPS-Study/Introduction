# ALPS C언어 스터디 2021

> 스터디 강사 : ALPS 노정훈 (컴퓨터학과 20)<br>스터디 일시 : 미정

## 수업 목표

컴퓨터와 C언어에 대한 사전지식 없이 컴퓨터프로그래밍I 수업을 이수해야하는 신입생, 또는 전공에 필요한 C언어를 다시 점검하고 싶은 재학생을 대상으로 한 스터디입니다. 본 스터디에서는 알고리즘과 C언어에 필요한 간단한 컴퓨터 구조를 배우고, C언어의 기본을 배우는 것을 목표로 합니다.

|  7   |         구조체 연산, 별칭, 파일 입출력         |
